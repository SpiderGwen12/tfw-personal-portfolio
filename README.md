# 🌱 Farmspherica – Smart Hydroponics Towards Greener Tomorrow

## Overview
Farmspherica Innovations Pvt. Ltd. is dedicated to building a sustainable future through smart hydroponics technology.  
This website highlights our vision, features, and solutions for next‑gen gardening, combining innovation with eco‑friendly practices.

## ✨ Features
- **Hero Section**: Prominent brand logo, tagline, and call‑to‑action buttons
- **Interactive Flip Cards**: Hover/tap animations that reveal key benefits (automation, water savings)
- **Responsive Design**: Optimized for desktop and mobile devices
- **Sustainability Focus**: Emphasizes eco‑friendly gardening and water conservation

## 🛠 Tech Stack
- **HTML5** – Structure and content
- **CSS3** – Styling, animations, and responsive layout
- **JavaScript (ES6)** – Interactivity (flip cards, mobile tap support)
- **GitHub Pages / Vercel** – Hosting and deployment
